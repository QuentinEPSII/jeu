Super Luffy Bros avec Phaser.io

But du jeu

Finissez les trois niveaux en recupérant ses 3 repas préférés,
Dans le dernier niveau des bonus sont disponibles, ils vous seront peut-être utiles.

Ne pas oublier

Luffy ne sait pas nager alors evitez l'eau si vous ne voulez pas recommencer le niveau !
Exceptionnelement il vous est possible de prendre deux fruits du démon (les bonus) sans mourir.

Commandes

Pour avancer : "D" ou "→",
Pour reculer : "Q" ou "←",
Pour Sauter : "Z" ou "↑",
Pour interagir avec les portes "Espace".
